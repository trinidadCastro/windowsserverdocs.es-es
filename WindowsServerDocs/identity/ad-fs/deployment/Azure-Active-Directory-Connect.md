# <a name="azure-active-directory-connect"></a>Conectar de Azure Active Directory

>Se aplica a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Azure AD Connect integrará los directorios local con Azure Active Directory. Esto te permite proporcionar una identidad común para los usuarios para aplicaciones de Office 365, Azure y SaaS integrados con Azure AD. .  
  
Para obtener más información, consulta [integrar las identidades local con Azure Active Directory](https://azure.microsoft.com/en-us/documentation/articles/active-directory-aadconnect/).
